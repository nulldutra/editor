## Installation

To install the configuration, clone this repository to `~/.config/nvim`

```sh
git clone https://github.com/bsdutra/nvim-ide ~/.config/nvim
```

### Installing Neovim on ArchLinux

```sh
pacman -S neovim
```

### Installing Neovim on FreeBSD

```sh
pkg install neovim
```
